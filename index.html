Java, C++, NLP, Machine Learning, System Architecture, DevOps, Agile:Scrum/XP

Unicorn Burrito as we named it - in homage to time cube theory - was a chat bot project. Cheryl-Engine

Open Source | 2013 - 2015
C++, OpenGL, Systems Architecture
(Solo Project, Personal)

Cheryl Engine is an OpenGL game engine that I had expanded from Blit3D. Through writing and revising the, now abandoned, engine I was able to learn a lot about OpenGL and systems architecture. On the last revision I set out to support multi-platform building. After making a lot of headway I abandoned it while working on the logging utilities. As it turns out, stack tracing is easier said than done in such a manner. Modifying some assembly in the StackWalker library I was able to support most builds, but in the end my efforts were unable to backtrace the stack on x86 gcc windows builds.
